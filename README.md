# CareSplit Backend

TypeScript backend API for CareSplit application.

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Start (Production)
```bash
npm start
```

## Project Structure
- `src/` - TypeScript source files
- `dist/` - Compiled JavaScript output

## Technologies
- Express.js
- TypeScript
- Node.js
